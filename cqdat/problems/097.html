Introduction


Extensible Markup Language (XML) is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable. It is used heavily in the IT world to pass data around from system to system.

XML looks very much like HTML in that XML elements are encased in a set of less than and greater than signs. End tags are similar but have a slash at the beginning of the element name. Here is an example:


XML tags can be named whatever you want, and they can be nested (which is what starts to unleash the true power of XML). Your job is to look through an XML file and list the element names that you find in order and the number of times that element name exists in the XML file. Remember - do not count the ending tags, only the starting ones!
